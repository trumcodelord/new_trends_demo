# Bảng điều khiển phát hiện xu hướng tin tức

## Cài đặt

```bash
pip install -r requirements.txt
```

## Chạy dashboard

```bash
streamlit run app.py
```

Mở trình duyệt tại:

```text
http://localhost:8501
```

## File chính

- `app.py`: giao diện Streamlit
- `trends.json`: dữ liệu dashboard
- `clusters.xlsx`: kết quả gom cụm
- `news-trend-detection.ipynb`: notebook xử lý chính
